# План выполнения работы по автоматизации тестирования

## Позитивные сценарии

### Сценарий 1: Заполнение формы «Кредит по данным карты» со статусом карты «APPROVED»

1. Нажать на кнопку «Купить в кредит».
2. Заполнить поля формы «Кредит по данным карты» валидными значениями:
    - **Номер карты**: карта со статусом «APPROVED» (например: 5555 5555 5555 5556).
    - **Номер месяца**: цифры от 01 до 12 (например: 07).
    - **Год**: последние две цифры текущего или будущего года (например: 27).
    - **Владелец**: имя на латинице (например: Elena Ivanova).
    - **CVC/CVV**: трёхзначный код (например: 456).
3. Нажать на кнопку «Продолжить».
4. **Ожидаемый результат**: Появление сообщения об успешной операции. Форма успешно отправлена. В базе данных регистрируется новая транзакция со статусом "APPROVED".
